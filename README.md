## Setup

To run this project, install it by download or clone.

#### Api key
- You need first to register new account in (https://unsplash.com/) to get an api key, 
Then embed it to the project to run the app.

#### System requirements
- Android studio v4.1.1
- Minimum sdk v21
- Build tool v29.0.3
- Compile sdk v29
